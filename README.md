# inventory-mgmt-app-py

## Setup

Download the db/products.csv and save it as `data/products.csv`.

## Prerequisites

This application assumes you have installed Python 3.x and corresponding Pip. It may work, but is untested on Python 2.x.

## Installation

Download the source code:

git clone git@github.com:ampesce19/inventory-mgmt-app-py.git


## Usage

Run the application:

```shell
python3 app/products_app.py
```
